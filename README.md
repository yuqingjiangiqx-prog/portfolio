# 个人品牌网站 Portfolio

一个用于求职和个人品牌展示的 React + Tailwind 单页网站，定位为“用户增长与运营体系搭建专家”。

## 技术栈

- React
- Vite
- Tailwind CSS
- Framer Motion
- lucide-react

## 运行方式

```bash
npm install
npm run dev
```

浏览器打开终端显示的本地地址即可访问。

## 项目结构

```text
.
├── index.html
├── package.json
├── postcss.config.js
├── tailwind.config.js
├── vite.config.js
└── src
    ├── App.jsx
    ├── index.css
    └── main.jsx
```

## 页面模块

- Header 导航
- Hero 首屏
- 关键数据区
- 我能解决什么问题
- 精选案例，含案例详情弹层
- 关于我
- 能力体系
- 工具与学习，轻量 AI 展示
- 联系我
- Footer

## 可替换内容

- 邮箱：`src/App.jsx` 中搜索 `yourname@example.com`
- 微信：`src/App.jsx` 中搜索 `WeChat ID 待补充`
- 案例结果：`src/App.jsx` 中搜索 `待补充`
- 名字或 Logo：`src/App.jsx` 中搜索 `Growth Operator`
