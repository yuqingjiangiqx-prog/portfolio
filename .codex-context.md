## Project

- A personal brand / portfolio single page site for job search and personal
  positioning.
- Current positioning in README: "用户增长与运营体系搭建专家".
- Main user-facing language is Chinese.

## Stack

- React 19
- Vite 6
- Tailwind CSS 3
- Framer Motion
- lucide-react

## Commands

- Install dependencies: `npm install`
- Start dev server: `npm run dev`
- Build production assets: `npm run build`
- Preview build: `npm run preview`

- `package.json`: scripts and dependencies.

## Token-Saving Rules

- Start by reading this file, then inspect only files related to the task.
- Avoid scanning `node_modules`, `dist`, `.npm-cache`, and `package-lock.json`
  unless dependency or build output details are directly relevant.
- Prefer targeted PowerShell searches when `rg` is unavailable. On this machine,
  `rg.exe` returned "Access is denied" during setup.
- When reading large component files, inspect imports, exported component names,
  state/data sections, and the specific JSX/CSS area involved before reading the
  whole file.
- Keep command output short. For failed builds/tests, summarize the error and
  only open the relevant lines.
